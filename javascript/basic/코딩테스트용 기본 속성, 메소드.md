**정렬**

- **배열 정렬 : 배열.sort([비교 함수])**
    - **sort()는 기본적으로 유니코드 값 순서대로 정렬을 한다.**
    - **sort() 메소드는 원본 배열을 변경하는 함수이며, 동시에 변경된 배열을 리턴하기 때문에 특정 변수에 sort된 배열 값을 저장할 수 있다.**
    - **따라서, 오름차순, 내림차순 등으로 정렬을 하기 위해서는 매개변수로 정렬 순서를 정의하는 함수를 작성해야 한다.**
    
    **<쉽게 사용하는 방법>**
    
    - **오름차순 : arr.sort((a, b) => a-b)**
    - **내림차순 : arr.sort((a, b) => b-a)**
    - **비교 함수는 ((이전 값, 현재 값) => { 정렬 가중치 함수 }) 형태로 이루어짐.**
        - 비교 함수가 생략되면, 배열의 element들은 문자열로 취급되어, 유니코드 값 순서대로 정렬됩니다.
        - 비교 함수는 두 개의 배열 element를 파라미터로 입력받습니다.
        - 이 함수가 a, b 두개의 element를 파라미터로 입력받을 경우,
        - 이 함수가 리턴하는 값이 0보다 작을 경우,  a가 b보다 앞에 오도록 정렬하고,
        - 이 함수가 리턴하는 값이 0보다 클 경우, b가 a보다 앞에 오도록 정렬합니다.
        - 만약 0을 리턴하면, a와 b의 순서를 변경하지 않습니다.
    
    ```jsx
    <오름차순 정렬>
    
    const arr = [2, 1, 3, 10];
    
    <어려운 방법>
    arr.sort(function(a, b) {
    	if(a < b) return -1; // a가 b보다 앞에 오도록 정렬
    	if(a > b) return 1; // a가 b보다 뒤에 오도록 정렬
    	if(a === b) return 0; // a와 b의 순서 변경x
    });
    
    <쉬운 방법>
    arr.sort((a, b) => a-b); // 오름차순
    arr.sort((a, b) => b-a); // 내림차순
    
    console.log(arr) // 오름차순 : [1, 2, 3, 10] / 내림차순 : [10, 3, 2, 1]
    ```

### ***모든 요소에 함수를 적용한 새로운 배열 반환***

- **forEach, map, filter, reduce 메서드를 적용한 배열은 새 변수에 할당해야 함. (원본 배열은 변경되지 않음.)**
- **for 루프를 스킵할 수 있어 편함**

- **배열.forEach( function (값, 인덱스) {…} )**
    - **모든 요소에 단순히 콜백 함수 실행(새로운 배열 생성×) : 배열.forEach((v, i) ⇒ {콜백함수})**

- **배열.map( function (값) {…} )**
    - **모든 요소에 콜백 함수 적용하여 새로운 배열 리턴**

```jsx
let numbers = [1, 2, 3];
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]
```

- **배열.filter( function (값) {…} )**
    - **모든 요소에 콜백 함수 적용하여 조건을 통과한 요소들만 모아 새로운 배열 반환 :**

```jsx
let numbers = [1, 2, 3, 4, 5];
let evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // Output: [2, 4]
```

- **배열.reduce( function(누산기, 값, 인덱스, 배열) {…}, 초기값)**

    - **여러 개의 값이 담긴 배열이 줄어서 최종적으로 하나의 값으로 만드는 과정**

# 5. 집합/맵 자료구조

- **집합(Set)**은 JavaScript의 내장 객체로, 배열과 비슷하지만 **같은 값을 중복 저장하지 않는다**는 중요한 차이가 있음.
- 배열의 사촌이라고 생각하면 편함.
- 집합이라는 이름답게 합집합,

### 속성

- **집합 길이 : 집합.size**

### 메소드

**집합 생성 / 초기화 / 배열 변환**

- **새 집합 생성 : new Set()**
- **배열로 집합 초기화 : new Set([1, 2, 3, 4, 5])**
- **집합을 배열로 변환 : Array.from(집합)**
- **새로운 집합 복사 : const 새집합 = new Set(원래 집합**)
- **집합의 요소 유무 불린 반환 : 집합.has(요소)**

---

**집합 요소 추가 / 제거**

- **집합 요소 추가 : 집합.add()**
- **집합 요소 제거 : 집합.delete()**
- **집합의 모든 요소 제거 : 집합.clear()**

---

 **수학적 집합 메소드 구현 방법**

- **합집합 구현**

```jsx
function setUnion(arr1, arr2) {
	const set1 = new Set(arr1);
	const set2 = new Set(arr2);
	return Array.from(new Set([...arr1, ...arr2]));
}
```

- **교집합 구현**

```jsx
function setIntersection(arr1, arr2) {
    const set1 = new Set(arr1)
    const set2 = new Set(arr2)
    const intersectionSet = new Set();
    
    for(let element of set1) {
        if(set2.has(element)) {
            intersectionSet.add(element);
        }
    }
    
    return Array.from(intersectionSet);
}
```
